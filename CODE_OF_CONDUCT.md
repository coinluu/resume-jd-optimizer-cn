# 社区行为准则

我们希望社区保持友善、专业、尊重隐私，并让不同求职背景的人都能安全参与。

## 鼓励的行为

- 针对内容和规则讨论，不攻击个人。
- 尊重应届、转行、空窗和不同教育背景的求职者。
- 对不确定结论明确说明证据和边界。
- 主动提醒案例脱敏，保护候选人和公司的隐私。
- 接受建设性反馈，并愿意修正错误。

## 不可接受的行为

- 歧视、骚扰、侮辱、人身攻击或恶意嘲讽。
- 公开他人真实简历、联系方式、身份信息或商业机密。
- 鼓励伪造经历、证书、数据、职级或背调信息。
- 利用社区内容进行未经说明的营销、引流或收费推销。
- 重复发布垃圾内容或故意干扰讨论。

## 处理方式

维护者可以编辑、隐藏或删除违反本准则的内容，并视情况限制参与权限。严重的隐私泄露或安全问题应通过 [SECURITY.md](SECURITY.md) 中的私下渠道报告。

参与本项目即表示同意遵守本准则。
